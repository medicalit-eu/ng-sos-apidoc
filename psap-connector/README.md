# NG SOS PSAP Connector Server

## Overview

The NG SOS PSAP Connector Server provides connectivity and integration between the NG SOS platform and Public Safety Answering Points (PSAPs).

## API Documentation

This folder contains the OpenAPI specification for the PSAP Connector Server API.

- **Version**: 2.0.0
- **Contact**: Martin Dybal (mdybal@medicalit.eu)

## API Specification

The complete API specification is available in the [openapi.yaml](./openapi.yaml) file (YAML format for enhanced readability and maintainability).

You can also view the interactive API documentation on our [API documentation website](https://medicalit-eu.github.io/ng-sos-apidoc/).

## Key Features

- PSAP integration and connectivity
- Emergency call routing
- System information and about endpoints
- Secure authentication and authorization
