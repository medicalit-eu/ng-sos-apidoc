# NG SOS Emergency Message Server (EMS)

## Overview

The NG SOS Emergency Message Server (EMS) is a critical component of the NG SOS platform that handles emergency message processing and management.

## API Documentation

This folder contains the OpenAPI specification for the Emergency Message Server API.

- **Version**: 1.0.0
- **Contact**: Martin Dybal (mdybal@medicalit.eu)

## API Specification

The complete API specification is available in the [openapi.json](./openapi.json) file.

You can also view the interactive API documentation on our [API documentation website](https://medicalit-eu.github.io/ng-sos-apidoc/).

## Key Features

- SMS reception and processing via Develogi integration
- Emergency message handling
- Integration endpoints for various emergency services
