# Portal URL API

## Overview

The Portal URL API provides URLs and endpoints for the NG SOS portal, facilitating easy integration into third-party applications.

## API Documentation

This folder contains the OpenAPI specification for the Portal URL API.

- **Version**: 1.0.0
- **Contact**: mdybal@medicalit.eu

## API Specification

The complete API specification is available in the [openapi.json](./openapi.json) file.

You can also view the interactive API documentation on our [API documentation website](https://medicalit-eu.github.io/ng-sos-apidoc/).

## Key Features

- Incident list overview
- Incident creation with pre-filled forms
- Incident detail viewing
- Incident archive with filtering capabilities
